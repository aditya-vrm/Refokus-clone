Refokus Website Clone 🚀

A fully responsive frontend clone of the Refokus website, built using React to practice modern UI development, component structuring, and smooth user interactions.

This project focuses on recreating a pixel-perfect UI, animations, and layout behavior inspired by the original Refokus website.
⚠️ This clone is created strictly for educational and learning purposes.

🔗 Live Demo

👉 https://refokus-clone-gamma-liard.vercel.app/

🛠 Tech Stack

React.js (Functional Components)

JavaScript (ES6+)

CSS / Tailwind-style utility classes

Modern UI Animations

Responsive Design

✨ Features

🎯 Pixel-perfect Refokus-inspired UI

📱 Fully responsive (Mobile, Tablet, Desktop)

⚡ Smooth hover & scroll animations

🧩 Reusable and clean React components

🎨 Modern layout & typography

🚀 Fast and optimized frontend performance
